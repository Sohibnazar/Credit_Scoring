# Модель оценки вероятности дефолта (PD) по потребительским кредитам

Данный проект посвящён построению модели машинного обучения для оценки
вероятности дефолта по потребительским кредитам на основе данных о клиенте
и параметрах кредита.

Проект выполнен в рамках учебного курса по классическому машинному обучению.

---

## 🎯 Цель проекта

Цель проекта — разработать модель бинарной классификации, которая прогнозирует
вероятность дефолта клиента (`loan_status`) и может быть использована
для предварительной оценки кредитного риска.

---

## 📊 Описание данных

Используется публичный кредитный датасет, который **не применялся в ходе обучения**.

Характеристики датасета:
- Количество строк: **более 3000**
- Количество признаков: **более 7**
- Целевая переменная: `loan_status`
  - `0` — кредит без дефолта  
  - `1` — дефолт по кредиту  

### Основные признаки:
- `person_age` — возраст клиента  
- `person_income` — годовой доход  
- `person_home_ownership` — тип владения жильём  
- `person_emp_length` — стаж работы (лет)  
- `loan_amnt` — сумма кредита  
- `loan_int_rate` — процентная ставка  
- `loan_intent` — цель кредита   
- `cb_person_default_on_file` — наличие дефолтов в прошлом  
- `cb_person_cred_hist_length` — длина кредитной истории  

---

## 🧠 Используемая модель

- Алгоритм: **Logistic Regression**
- Тип задачи: **бинарная классификация**
- Метрика качества: **ROC-AUC**

### Результат:
- ROC-AUC на тестовой выборке: **0.85**

Данный результат свидетельствует о высокой способности модели
различать дефолтные и недефолтные кредиты.
